Setup Scripts

Helper scripts are available via HTTP for download to your domain controller and certificate server.

Access Setup Scripts

Scripts are served at:

Available scripts:

  • /share/configure-domain-controller.ps1
  • /share/configure-certificate-server.ps1

Download Instructions

From PowerShell:

Invoke-WebRequest -Uri "http://<server-ip>:5000/share/configure-domain-controller.ps1" -OutFile "configure-domain-controller.ps1"

From browser:

Navigate to the URL above and download the scripts directly.

Note: Replace localhost with your server's IP address when accessing from other machines.

Cloudflare Authentication

Authenticate with Cloudflare to authorize this application to manage tunnels on your account.

Note: When you click the login link, Cloudflare will ask you to select a zone for authorization. This is just for OAuth authorization purposes - it does not create a tunnel or affect your existing tunnels. You can select any zone you manage. After authentication, you'll be able to reuse existing tunnels or create new ones.
